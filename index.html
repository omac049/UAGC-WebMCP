Technical appendix

For engineering and compliance partners — expand when needed.

What is WebMCP in one paragraph?

WebMCP is a browser API (W3C Web Machine Learning CG; Chrome early preview) for registering JavaScript "tools" with names, descriptions, and schemas so AI assistants can call them instead of clicking around. It complements backend MCP: same philosophy, different surface — the live page you own.

Example: registering a tool

Illustrative — API names may evolve with the spec.

navigator.modelContext.registerTool({
  name: "searchPrograms",
  description: "Search UAGC programs by keyword, level, modality",
  inputSchema: {
    type: "object",
    properties: {
      keyword: { type: "string", description: "e.g. 'education'" },
      level: { type: "string", enum: ["associate","bachelor","master"] },
      modality: { type: "string", enum: ["online","hybrid"] }
    },
    required: ["keyword"]
  },
  execute: async ({ keyword, level, modality }) => {
    const results = await ProgramService.search({ keyword, level, modality });
    return { programs: results.map(p => ({
      id: p.id, name: p.name, level: p.level, url: p.url
    }))};
  }
});
